# Scale & Sell

Scale & Sell is the premier marketplace for buying and selling verified, revenue-generating businesses. We provide secure escrow services, expert support, and transparent processes to connect business buyers and sellers.

## Project Info

This project is a modern web application built to facilitate the buying and selling of established businesses.

## Getting Started

To get started with the project locally, follow these steps:

1.  **Clone the repository:**

    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Navigate to the project directory:**

    ```sh
    cd scale-sell-hub
    ```

3.  **Install dependencies:**

    ```sh
    npm install
    ```

4.  **Start the development server:**
    ```sh
    npm run dev
    ```

## Technologies Used

This project is built using the following technologies:

- **Vite**: Next Generation Frontend Tooling
- **TypeScript**: Typed JavaScript at Any Scale
- **React**: A JavaScript library for building user interfaces
- **shadcn/ui**: Reusable components built with Radix UI and Tailwind CSS
- **Tailwind CSS**: A utility-first CSS framework
- **Supabase**: Open Source Firebase Alternative (Backend & Auth)

## Features

- **Browse Listings**: View a wide range of verified businesses for sale.
- **Sell Your Business**: List your business and reach potential buyers.
- **Secure Transactions**: Integrated escrow services for safe deals.
- **User Dashboards**: Dedicated dashboards for Investors, Sellers, and Admins.
- **Real-time Messaging**: Communicate directly with other users.

## Deployment

To deploy the application, build the project using:

```sh
npm run build
```

Then deploy the `dist` folder to your preferred hosting provider (e.g., Vercel, Netlify, AWS).
